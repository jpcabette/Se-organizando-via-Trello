# Se-organizando-via-Trello
Como estruturar um fluxo de trabalho via Trello, a fim de gerenciar prazos, audiências e tarefas como advogado. 

'Etiquetas': Crie um Lista com cartões coloridos por tipo (ex: vermelho = Prazos, azul = Audiências, verde = Atendimento/Diligência, amarelo = Prospecção novos casos, roxo = Social, laranja = Desenvolvimento/Estudos).

Crie uma Lista chamada 'Hoje': Adicione as tarefas para conclusão no dia atual.

Crie uma Lista chamada 'Ordem': Compromissos futuros, ordenados do presente ao longínquo.

Crie uma Lista chamada 'Completo': Para as tarefas finalizadas.

Nos cartões, inserir na opção 'Capa', a cor correspondente da Lista 'Etiqueta' para a tarefa ali incluída: Por exemplo, se o novo cartão for referente a uma audiência, ele deve ter a cor 'Azul'.

Ainda em relação aos cartões, seu título deve ter as informações relevantes: Por exemplo: [AUDIÊNCIA XX/XX/20XX - 12H00] FULANO DE TAL X CICLANO DE TAL - PROCESSO N.º XXXXXXX-XX.XXXX.X.XX.XXXX .
Todos os cartões devem ter inseridas as datas para a sua conclusão/atualização, com o uso da ferramenta de agenda.
Para tarefas complexas usar os 'checklists' para separação das etapas.

Eficiência total! 🚀
